> 이것은 형식을 보여주는 **예시 메모리(참고본)**입니다 — 라이브 메모리에 적재되지 않습니다.
> 한 파일 = 한 사실. 인덱스(MEMORY.md)는 각 파일의 `description` 으로 **자동 생성**됩니다.
> 본인 프로필은 이 형식대로 `~/.claude/cc-memory/` 에 새 파일로 직접 작성하세요.

나는 누구인가: <역할/직무>. 기술 수준: <비개발자 / 개발자>. 선호: <간결한 보고 · 확인 후 실행 등>.

관련 메모리는 [[example_feedback_concise]] 처럼 이중 대괄호로 링크합니다(아직 없는 이름이어도 됨 — 나중에 쓸 거리 표시).
