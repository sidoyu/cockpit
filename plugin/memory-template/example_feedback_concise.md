> 이것은 **예시 메모리**입니다(type: feedback). feedback·project 메모리는 본문에 **Why / How to apply** 를 함께 적습니다.

(예시) 보고는 결론부터, 군더더기 없이.

**Why:** 사용자가 긴 세션에서 핵심만 빠르게 보길 원함.
**How to apply:** 응답을 결론 → 근거 순으로, 불필요한 서론·반복 제거. 선택지에는 추천을 함께 명시. 관련: [[example_user_profile]].
